About F1World

Our Mission

F1World is a machine learning project that aims to predict Formula 1 race outcomes using various statistical and machine learning models. Our project utilizes real F1 race data to train and test different models including:

  • Simple Linear Regression (SLR)
  • Multiple Linear Regression (MLR)
  • Logistic Regression
  • Polynomial Regression
  • K-Nearest Neighbors (KNN)

Our models analyze various factors including weather conditions, track temperature, driver experience, car specifications, and more to make predictions about race outcomes.
